## 评分模型概述

本评分模型用于评估热点话题与品牌传播需求的匹配度，通过五个维度的评分帮助筛选最适合的选题。

### 评分原则
- 每个维度评分范围为 0-10 分
- 评分基于定量数据（热度指数）和定性分析（品牌契合度）
- 评分需结合品牌调性、地区特点、行业特点、传播目标进行综合判断
- 话题分类器自动为每个热点打上地区和行业标签

### 权重配置（默认）
- 相关性评分：20%（权重 0.2）
- 地区相关性评分：15%（权重 0.15）
- 行业相关性评分：15%（权重 0.15）
- 火爆价值评分：25%（权重 0.25）
- 长期传播价值评分：25%（权重 0.25）

### 聚焦模式权重调整
- 地区聚焦模式：地区相关性权重提升至 0.3，相关性降至 0.1
- 行业聚焦模式：行业相关性权重提升至 0.3，相关性降至 0.1
- 全面模式：使用默认权重

## 维度一：相关性评分

### 评分范围
0-10 分，分数越高表示热点与品牌关联度越强

### 评分标准

| 分数段 | 相关性描述 | 判断标准 |
|--------|------------|----------|
| 9-10分 | 极强相关 | 热点话题直接涉及品牌、产品或核心业务领域；关键词完全匹配 |
| 7-8分 | 强相关 | 热点话题与品牌调性高度契合；产品能直接借势 |
| 5-6分 | 中等相关 | 热点话题与品牌有一定关联，需要巧妙结合；可通过创意连接 |
| 3-4分 | 弱相关 | 热点话题与品牌关联度较低，需强扭话题；可能存在牵强附会风险 |
| 0-2分 | 无相关 | 热点话题与品牌完全无关；强行借势可能导致负面效果 |

### 评分要素

1. **品牌调性契合度**
   - 热点的价值观、情感基调是否与品牌一致
   - 热点的受众画像是否与目标用户重合

2. **产品关联度**
   - 产品是否可以作为热点话题的解决方案
   - 产品功能是否能在热点场景中得到展示

3. **核心价值匹配度**
   - 热点话题是否体现品牌的核心价值主张
   - 品牌能否在热点中提供独特价值

## 维度二：地区相关性评分

### 评分范围
0-10 分，分数越高表示热点与目标地区的关联度越强

### 评分标准

| 分数段 | 地区相关性描述 | 判断标准 |
|--------|----------------|----------|
| 9-10分 | 极强相关 | 热点话题直接发生在目标地区；地区关键词完全匹配 |
| 7-8分 | 强相关 | 热点话题涉及目标地区；地区影响明显 |
| 5-6分 | 中等相关 | 热点话题在目标地区有一定关注度 |
| 3-4分 | 弱相关 | 热点话题与目标地区关联度较低 |
| 0-2分 | 无相关 | 热点话题与目标地区无关；或热点为全国性话题但品牌专注本地 |

### 评分要素

1. **地区标签匹配**
   - 热点话题的地区标签是否包含目标地区
   - 话题是否在目标地区有较高讨论度

2. **地区受众影响**
   - 热点是否对目标地区用户有直接影响
   - 热点是否符合目标地区的文化特点

3. **本地化适配度**
   - 品牌是否能在该地区有效借势此热点
   - 热点是否适合在该地区传播

### 特殊情况处理
- 如果热点地区标签为"全国"但品牌专注于特定地区，地区相关性评分应降低
- 如果品牌目标地区为全国，则地区相关性评分参考全国话题的热度

## 维度三：行业相关性评分

### 评分范围
0-10 分，分数越高表示热点与所在行业的关联度越强

### 评分标准

| 分数段 | 行业相关性描述 | 判断标准 |
|--------|----------------|----------|
| 9-10分 | 极强相关 | 热点话题直接涉及品牌所在行业核心领域；行业关键词完全匹配 |
| 7-8分 | 强相关 | 热点话题与品牌所在行业高度相关；行业话题主流 |
| 5-6分 | 中等相关 | 热点话题与行业有一定关联，需要巧妙结合 |
| 3-4分 | 弱相关 | 热点话题与行业关联度较低 |
| 0-2分 | 无相关 | 热点话题与行业完全无关 |

### 评分要素

1. **行业标签匹配**
   - 热点话题的行业标签是否包含品牌所在行业
   - 话题是否涉及行业核心议题

2. **行业影响力**
   - 热点是否对行业有重要影响
   - 热点是否反映行业趋势

3. **专业适配度**
   - 品牌是否能在行业中提供专业见解
   - 热点是否适合在行业圈层传播

### 特殊情况处理
- 如果热点没有行业标签，需根据话题内容判断是否与行业相关
- 如果品牌属于多个行业，需综合评估所有行业的关联度

---

## 维度四：火爆价值评分

### 评分范围
0-10 分，分数越高表示热点传播潜力越大

### 评分标准

| 分数段 | 火爆价值描述 | 判断标准 |
|--------|--------------|----------|
| 9-10分 | 爆款级别 | 多平台热搜Top1；热度指数超过千万；讨论量巨大 |
| 7-8分 | 高热度 | 单平台热搜Top3；热度指数数百万；传播速度快 |
| 5-6分 | 中等热度 | 平台热搜前10；热度指数数十万；有一定传播范围 |
| 3-4分 | 低热度 | 热点指数较低；仅在特定圈层传播；时效性短 |
| 0-2分 | 无热度 | 非热点话题；传播范围极小 |

### 评分要素

1. **热度指数**
   - 阅读量、讨论量、搜索量的综合表现
   - 各平台热度排名位置

2. **传播速度**
   - 热点话题的上升趋势
   - 短时间内的爆发能力

3. **平台覆盖度**
   - 热点在多个平台的同步情况
   - 跨平台传播的广度

---

## 维度五：长期传播价值评分

### 评分范围
0-10 分，分数越高表示热点可持续传播价值越大

### 评分标准

| 分数段 | 长期价值描述 | 判断标准 |
|--------|--------------|----------|
| 9-10分 | 高持续价值 | 话题可持续数周/数月；具备深度挖掘空间；可形成系列内容 |
| 7-8分 | 较高持续价值 | 话题可持续数天；有二次创作空间；适合多角度切入 |
| 5-6分 | 中等持续价值 | 话题可持续1-2天；有一定延伸性；可做单次传播 |
| 3-4分 | 低持续价值 | 话题时效性强，迅速消退；延伸空间有限 |
| 0-2分 | 无持续价值 | 瞬时热点，无传播延展性 |

### 评分要素

1. **话题持续性**
   - 热点话题的生命周期长度
   - 话题是否具备持续讨论的潜力

2. **二次创作空间**
   - 热点话题是否适合多角度解读
   - 是否能衍生出系列内容

3. **品牌正向影响**
   - 借势传播是否对品牌形象有正向提升
   - 是否存在负面风险

## 综合得分计算

### 默认权重计算公式
```
综合得分 = (相关性评分 × 0.2) + (地区相关性评分 × 0.15) + (行业相关性评分 × 0.15) + (火爆价值评分 × 0.25) + (长期传播价值评分 × 0.25)
```

### 得分解读

| 综合得分范围 | 推荐等级 | 行动建议 |
|--------------|----------|----------|
| 8.0-10.0分 | 强烈推荐 | 立即策划内容，快速跟进 |
| 6.0-7.9分 | 推荐 | 纳入选题池，择机执行 |
| 4.0-5.9分 | 可选 | 视资源情况考虑 |
| 0-3.9分 | 不推荐 | 放弃该热点 |

## 聚焦模式权重调整

### 地区聚焦模式
**适用场景**：品牌目标地区明确，需要精准匹配本地热点

**权重配置**：
- 相关性评分：10%（权重 0.1）
- 地区相关性评分：30%（权重 0.3）
- 行业相关性评分：15%（权重 0.15）
- 火爆价值评分：25%（权重 0.25）
- 长期传播价值评分：20%（权重 0.2）

**计算公式**：
```
综合得分 = (相关性评分 × 0.1) + (地区相关性评分 × 0.3) + (行业相关性评分 × 0.15) + (火爆价值评分 × 0.25) + (长期传播价值评分 × 0.2)
```

### 行业聚焦模式
**适用场景**：品牌属于特定行业，需要聚焦行业热点

**权重配置**：
- 相关性评分：10%（权重 0.1）
- 地区相关性评分：15%（权重 0.15）
- 行业相关性评分：30%（权重 0.3）
- 火爆价值评分：25%（权重 0.25）
- 长期传播价值评分：20%（权重 0.2）

**计算公式**：
```
综合得分 = (相关性评分 × 0.1) + (地区相关性评分 × 0.15) + (行业相关性评分 × 0.3) + (火爆价值评分 × 0.25) + (长期传播价值评分 × 0.2)
```

### 全面模式（默认）
**适用场景**：需要综合评估热点的各个维度

**权重配置**：使用默认权重配置

## 评分示例

### 示例1：科技品牌新品推广（全面模式）

**背景信息**：
- 品牌名：XX科技
- 产品：智能手表Pro
- 行业关键词：智能穿戴、健康监测、运动科技
- 传播目标：产品曝光

**热点话题**："2024年最值得期待的健康科技产品"

**话题分类**：
- 地区：全国
- 行业：科技

**评分过程**：
1. 相关性评分：9分（直接涉及健康科技，与智能手表高度相关）
2. 地区相关性评分：5分（全国话题，品牌目标为全国）
3. 行业相关性评分：10分（科技行业，完全匹配）
4. 火爆价值评分：7分（科技圈热度较高，有讨论度）
5. 长期传播价值评分：8分（可做系列测评，适合持续推广）

**综合得分**：9×0.2 + 5×0.15 + 10×0.15 + 7×0.25 + 8×0.25 = 1.8 + 0.75 + 1.5 + 1.75 + 2 = **7.8分**

**推荐等级**：推荐
