## 输出结构概述

热点选题策略助手的输出采用结构化格式，包含风险预警（如有）、标题信息和详细的选题建议列表。

### 顶层信息字段
- **风险预警**（可选）：未来7天特殊节点风险预警
- **执行摘要**：简要说明本次选题推荐的核心发现
- **输入信息回顾**：品牌、产品、行业关键词、目标地区、传播目标、平台偏好、聚焦模式
- **数据来源**：热点数据来源平台、热点池规模
- **评分标准**：使用的评分维度和权重配置

### 选题建议列表（11个字段）
每个选题建议包含以下字段：
1. **选题方向**
2. **推荐渠道**
3. **传播契合点**
4. **执行建议**
5. **综合得分**
6. **相关性得分**
7. **地区相关性得分**
8. **行业相关性得分**
9. **火爆价值得分**
10. **长期传播价值得分**
11. **风险提示**

---

## 输出字段详细说明

### 1. 选题方向

**字段说明**：基于热点话题生成的具体内容方向

**包含内容**：
- **标题**：吸引眼球的传播标题
- **核心观点**：内容的核心论点或主张
- **内容形式**：建议采用的内容形式（图文、视频、直播等）

**示例**：
```
选题方向：
标题：当健康监测成为标配，智能手表如何重新定义你的生活
核心观点：智能手表不仅是配饰，更是健康管理的必备工具
内容形式：深度图文 + 产品测评视频
```

---

### 2. 推荐渠道

**字段说明**：建议发布内容的平台及理由

**包含内容**：
- **主推平台**：最重要的发布渠道
- **辅助平台**：配合传播的次要渠道
- **选择理由**：为什么选择这些平台（受众匹配、传播效果等）

**示例**：
```
推荐渠道：
主推平台：微博（科技博主话题讨论活跃，传播速度快）
辅助平台：小红书（用户关注健康生活，种草效果好）
选择理由：微博适合话题引爆，小红书适合深度种草
```

---

### 3. 传播契合点

**字段说明**：热点与品牌关联的具体切入点

**包含内容**：
- **关联角度**：如何将热点与品牌/产品联系起来
- **价值主张**：品牌能在这个话题中提供什么价值
- **情感共鸣**：如何触动用户的情感需求

**示例**：
```
传播契合点：
关联角度：将"健康科技"热点与智能手表的健康监测功能结合
价值主张：XX科技智能手表，让健康管理更简单、更科学
情感共鸣：关注健康是每个人的刚需，智能手表让健康触手可及
```

---

### 4. 执行建议

**字段说明**：具体的内容创作和发布建议

**包含内容**：
- **创作方向**：内容创作的具体思路和角度
- **关键要素**：内容中必须包含的核心元素
- **发布时机**：最佳发布时间
- **互动策略**：如何引导用户互动
- **风险提示**：需要注意的潜在风险

**示例**：
```
执行建议：
创作方向：从用户痛点切入，展示智能手表如何解决健康监测难题
关键要素：真实使用场景、数据对比、用户证言、专业背书
发布时机：工作日晚上8-10点，用户活跃度高
互动策略：发起话题讨论，邀请用户分享健康管理故事
```

---

### 5. 综合得分

**字段说明**：该选题的综合评分结果

**示例**：
```
综合得分：7.8分
推荐等级：推荐
```

---

### 6. 相关性得分

**字段说明**：品牌调性、产品关联度评分

**评分范围**：0-10分

**示例**：
```
相关性得分：9分
评价：直接涉及健康科技，与智能手表高度相关
```

---

### 7. 地区相关性得分

**字段说明**：与目标地区的关联度评分

**评分范围**：0-10分

**示例**：
```
地区相关性得分：5分
评价：全国话题，品牌目标为全国
```

---

### 8. 行业相关性得分

**字段说明**：与所在行业的关联度评分

**评分范围**：0-10分

**示例**：
```
行业相关性得分：10分
评价：科技行业，完全匹配
```

---

### 9. 火爆价值得分

**字段说明**：热度指数、传播速度评分

**评分范围**：0-10分

**示例**：
```
火爆价值得分：7分
评价：科技圈热度较高，有讨论度
```

---

### 10. 长期传播价值得分

**字段说明**：话题持续性、二次创作空间评分

**评分范围**：0-10分

**示例**：
```
长期传播价值得分：8分
评价：可做系列测评，适合持续推广
```

---

### 11. 风险提示

**字段说明**：需要注意的潜在风险，特别是高风险节点期间的营销风险

**包含内容**：
- **营销风险**：可能引发负面反应的内容
- **敏感点提示**：需要避免的敏感话题
- **合规建议**：遵守的平台规则和法律法规

**示例**：
```
风险提示：
营销风险：避免过度医疗化，强调辅助监测功能
敏感点提示：避免夸大产品效果，遵守广告法
合规建议：确保宣传内容真实可信，提供检测报告
```

## 完整输出示例

### 示例1：科技品牌新品推广（全面模式）

## 输入信息回顾

- 产品名：智能手表Pro
- 行业关键词：智能穿戴、健康监测、运动科技
- 目标地区：全国
- 传播目标：产品曝光
- 平台偏好：微博、小红书
- 聚焦模式：全面模式

## 数据来源
- 热点来源：微博热搜、百度热榜
- 热点池规模：50个热点话题
- 评分时间：2024-01-15 14:30:00

## 评分标准
- 相关性评分：20%
- 地区相关性评分：15%
- 行业相关性评分：15%
- 火爆价值评分：25%

## 风险预警格式

### 示例：高风险节点预警

```markdown
========================================
⚠️ 风险预警：未来7天特殊节点
========================================

节点1：九一八事变纪念日
- 日期：2024-09-18（还有5天）
- 风险级别：HIGH（高风险）
- 类型：特殊日期
- 建议主题：铭记历史、爱国、勿忘国耻、和平
- 规避建议：严禁娱乐化营销，保持严肃态度，避免不当言论

节点2：中秋节
- 日期：2024-09-17（还有4天）
- 风险级别：LOW（低风险）
- 类型：节日
- 建议主题：团圆、赏月、思乡、传统文化
- 规避建议：无特殊限制

========================================
```

## 不同场景的输出调整

### 场景1：地区聚焦模式

**调整策略**：
- 提高地区相关性评分的权重至30%
- 优先推荐与目标地区相关的话题
- 过滤与目标地区无关的地区性话题
- 报告头部标注"地区聚焦模式"

**输出示例**：
```markdown
# 热点选题策略建议（地区聚焦模式）

## 输入信息回顾
- 品牌名：XX餐饮
- 产品名：京味小吃
- 目标地区：北京
- 聚焦模式：地区聚焦模式

## 评分标准（地区聚焦模式）
- 相关性评分：10%
- 地区相关性评分：30%
- 行业相关性评分：15%
- 火爆价值评分：25%
- 长期传播价值评分：20%
```

---

### 场景2：行业聚焦模式

**调整策略**：
- 提高行业相关性评分的权重至30%
- 优先推荐与品牌行业相关的话题
- 过滤与品牌行业无关的话题
- 报告头部标注"行业聚焦模式"

**输出示例**：
```markdown
# 热点选题策略建议（行业聚焦模式）

## 输入信息回顾
- 品牌名：XX美妆
- 产品名：抗老精华
- 行业关键词：美妆、护肤、抗老
- 聚焦模式：行业聚焦模式

## 评分标准（行业聚焦模式）
- 相关性评分：10%
- 地区相关性评分：15%

- 火爆价值评分：25%
- 长期传播价值评分：20%
```

---

### 场景3：选题日历预警模式

**调整策略**：
- 在报告头部显示风险预警
- 每个选题建议增加特殊节点相关的风险提示
- 提供特殊节点的建议主题和规避建议

**输出示例**：
```markdown
# 热点选题策略建议

========================================
⚠️ 风险预警：未来7天特殊节点
========================================

节点：国家公祭日

- 建议主题：铭记历史、悼念、和平、爱国
- 规避建议：严禁娱乐化营销，保持庄重肃穆

========================================

## 选题建议 #1

### 风险提示
**特殊节点风险**：国家公祭日期间（12月13日），严禁娱乐化营销
**规避建议**：推迟娱乐类选题发布，保持庄重肃穆的态度
**替代方案**：改为品牌文化类内容，强调品牌社会责任

...
```
