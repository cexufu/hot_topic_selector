# 热点选题策略助手

## 任务目标
- 本 Skill 用于：帮助品牌、公关、市场、内容运营团队快速识别适合品牌传播的热点话题
- 能力包含：扫描全网热点池、地区/行业分类、多维度评分分析、选题日历预警、输出Top5选题建议及执行方案
- 触发条件：用户需要基于热点进行内容策划、营销传播、品牌推广等场景

## 前置准备
- 依赖说明：scripts脚本所需的依赖包及版本
  ```bash
  requests==2.31.0
  beautifulsoup4==4.12.2
  ```

## 操作步骤

### 标准流程

1. **获取用户输入信息**
   - 收集以下关键信息：
     - 品牌名：品牌或公司的正式名称
     - 产品名：需要推广的核心产品
     - 行业关键词：行业相关的3-5个关键词
     - 地区关键词：品牌目标地区（如北京、上海等，可选）
     - 传播目标：品牌曝光、产品转化、用户教育等
     - 平台偏好：微博、微信公众号、抖音、小红书等
     - 聚焦模式：地区聚焦模式/行业聚焦模式/全面模式（可选）
   - 基于用户输入信息，构建品牌画像和传播目标画像

2. **调用热点搜索工具获取全网热点池**
   - 执行命令：`python scripts/fetch_hot_topics.py --platforms weibo,baidu --limit 50`
   - 输出为JSON格式的热点列表，包含：话题标题、热度指数、来源平台、发布时间、地区分类、行业分类等
   - 如果需要指定平台或调整数量，使用相应参数：
     - `--platforms weibo,baidu,douyin`：指定数据源（默认：weibo,baidu）
     - `--limit 100`：返回热点数量（默认：50）
   - 热点数据自动包含地区和行业标签，便于后续筛选

3. **检查选题日历预警**
   - 调用选题日历管理器，检查未来7天的特殊节点
   - 如果存在高风险节点，在报告头部显示风险预警
   - 风险预警包含：节点名称、日期、风险级别、建议主题、规避建议
   - 示例：918事变纪念日（高风险）- 严禁娱乐化营销，保持严肃态度

4. **应用五维度评分模型对热点进行评分**
   - 读取 `references/scoring-guide.md` 了解评分标准
   - 对每个热点进行以下五个维度的评分（每个维度0-10分）：
     - 相关性评分：评估热点与品牌调性、产品、核心价值的关联程度（20%）
     - 地区相关性评分：评估热点与品牌目标地区的关联程度（15%）
     - 行业相关性评分：评估热点与品牌所在行业的关联程度（15%）
     - 火爆价值评分：评估热点的热度指数、传播速度、平台覆盖度（25%）
     - 长期传播价值评分：评估话题持续性、二次创作空间、品牌正向影响（25%）
   - 计算综合得分：综合得分 = (相关性×0.2) + (地区相关性×0.15) + (行业相关性×0.15) + (火爆价值×0.25) + (长期传播价值×0.25)
   - 根据聚焦模式调整评分权重：
     - 地区聚焦模式：提高地区相关性权重至30%
     - 行业聚焦模式：提高行业相关性权重至30%
     - 全面模式：使用默认权重
   - 按综合得分从高到低排序

5. **输出Top5选题建议及执行说明**
   - 读取 `references/output-format.md` 了解标准输出格式（11个字段）
   - 选择综合得分前5的热点，生成选题建议
   - 每个选题建议包含（11个字段）：
     - 选题方向：基于热点生成的标题和核心观点
     - 推荐渠道：根据平台偏好推荐的发布平台及理由
     - 传播契合点：热点与品牌关联的具体切入点
     - 执行建议：具体的创作方向、内容形式、发布时机等
     - 综合得分：该选题的综合评分结果
     - 相关性得分：品牌调性、产品关联度评分
     - 地区相关性得分：与目标地区的关联度评分
     - 行业相关性得分：与所在行业的关联度评分
     - 火爆价值得分：热度指数、传播速度评分
     - 长期传播价值得分：话题持续性、二次创作空间评分
     - 风险提示：需要注意的潜在风险（特别是高风险节点期间的营销风险）

6. **生成选题卡片（可选）**
   - 如果用户需要可下载的选题卡片，执行以下操作：
     1. 将每个选题建议保存为JSON格式，包含完整的11个字段和评分明细
     2. 调用 `scripts/card_generator.py` 生成选题卡片图片
     3. 执行命令：`python scripts/card_generator.py --input topic_1.json --output ./cards --index 1`
     4. 生成竖版卡片（800x1200px），包含所有11个字段的信息
     5. 批量生成Top5选题的5张卡片
   - 卡片特点：
     - 清晰的信息展示，无花哨设计
     - 包含品牌/产品信息、生成时间
     - 可直接下载使用，适合分享和存档

### 可选分支
- **地区聚焦模式**：当品牌目标地区明确时
  - 在用户输入中指定地区关键词
  - 提高地区相关性评分权重至30%
  - 优先推荐与目标地区相关的话题
  - 过滤与目标地区无关的地区性话题
  - 报告头部标注"地区聚焦模式"

- **行业聚焦模式**：当品牌属于特定行业时
  - 在用户输入中指定行业关键词
  - 提高行业相关性评分权重至30%
  - 优先推荐与品牌行业相关的话题
  - 过滤与品牌行业无关的话题
  - 报告头部标注"行业聚焦模式"

- **选题日历预警模式**：当需要提前规划特殊节点时
  - 自动检查未来7天的特殊节点
  - 如果有高风险节点，显示风险预警
  - 输出特殊节点信息和建议主题
  - 在选题建议中增加风险提示

- 当用户仅需要热点数据：仅执行步骤2，返回原始热点列表
- 当用户需要更多选题：调整评分权重或增加输出数量

## 注意事项
- 本Skill不包含舆情安全检查模块，如需评估话题风险，建议人工复核
- 选题日历预警功能仅提供风险提示，具体营销决策需结合实际情况
- 热点数据来源于公开API，可能存在延迟或不完整情况
- 话题分类器基于关键词匹配，如需提高识别准确率，可扩展关键词库
- 评分结果基于模型计算，需结合实际情况调整
- 执行建议需根据品牌调性和目标受众灵活调整
- 在高风险节点期间，严格遵循规避建议，避免不当营销
- 仅在需要时读取参考文档，保持上下文简洁

## 使用示例

### 示例1：品牌新产品推广
- **用户输入**：
  - 品牌名：XX科技
  - 产品名：智能手表Pro
  - 行业关键词：智能穿戴、健康监测、运动科技
  - 传播目标：产品曝光和用户教育
  - 平台偏好：微博、小红书
- **执行方式**：
  1. 调用 `scripts/fetch_hot_topics.py` 获取热点池
  2. 智能体分析智能手表与热点的关联性
  3. 输出Top5选题，重点关注健康、运动相关热点

### 示例2：品牌公关热点借势
- **用户输入**：
  - 品牌名：XX餐饮
  - 产品名：有机蔬菜套餐
  - 行业关键词：健康饮食、有机食品、绿色生活
  - 传播目标：品牌形象提升
  - 平台偏好：微信公众号
- **执行方式**：
  1. 调用脚本获取热点池
  2. 评分时提高长期传播价值权重
  3. 输出深度分析类选题建议

### 示例3：节日营销热点策划
- **用户输入**：
  - 品牌名：XX电商
  - 产品名：全品类商品
  - 行业关键词：购物节、促销、消费升级
  - 传播目标：销售转化
  - 平台偏好：抖音、小红书
- **执行方式**：
  1. 获取热点池，筛选节日相关热点
  2. 评分时提高火爆价值权重
  3. 输出带货导向的选题建议
